# vue-cursor

> Vue.js component


## Installation

``` bash
$ npm install --save vue-cursor
```


## Usage

``` vue
<template>
  <div class="app">
    <vue-cursor/>
  </div>
</template>

<script>
  import VueCursor from 'vue-cursor';
  export default {
    name: 'demo',
    components: {
      VueCursor
    }
  };
</script>
```

## Properties

* `blend-mode` **[String]**

  Optional; `difference` by defualt. The blend mode.
